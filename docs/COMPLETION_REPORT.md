## 使用说明

### 快速启动

1. 安装依赖
\`\`\`bash
cd backend && pip install -r requirements.txt
cd frontend && npm install
\`\`\`

2. 启动服务
\`\`\`bash
./start.sh
\`\`\`

3. 访问应用
- 前端: http://localhost:3000
- 后端: http://localhost:8000
- API文档: http://localhost:8000/docs
